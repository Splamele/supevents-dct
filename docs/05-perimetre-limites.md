# Périmètre et limites

## Périmètre fonctionnel

| Dans le périmètre          | Hors périmètre                      |
| -------------------------- | ----------------------------------- |
| Création d’événements      | Application mobile native           |
| Inscription à un événement | Paiement en cryptomonnaie           |
| Paiement via Stripe        | Intégration réseaux sociaux avancée |
| Gestion des utilisateurs   | Système de recommandation IA        |

## Limites techniques

| Limite technique                       | Justification                             |
| -------------------------------------- | ----------------------------------------- |
| Pas de migration de données existantes | Le système est développé from scratch     |
| Support limité aux navigateurs récents | IE11 est obsolète et non maintenu         |
| Pas de temps réel complexe (WebSocket) | Simplification de l’architecture initiale |
| Pas de stockage de données bancaires   | Conformité PCI-DSS assurée par Stripe     |
